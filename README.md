## 🛰️ Core Features

*   **🌌 3D Celestial Background**: Immersive space backgrounds rendered with high-performance Three.js particles and interactive planetary alignments.
*   **📡 Telemetry HUD Cursor**: A premium custom mouse cursor that tracks client telemetry coordinates, reacts dynamically to interactions, and displays contextual HUD state tags (e.g., `< FOLLOW >` for Instagram, `< CONNECT >` for LinkedIn).
*   **💫 Tech Constellation Map**: Interactive stack showcase organized into orbital telemetry tiers with technical parameter stats on hover.
*   **🌀 Symmetrical Connect Grid**: Symmetrical 2x2 social interface (Email, LinkedIn, GitHub, Instagram) balanced perfectly for sleek desktop visualization.
*   **🛸 Kinetic Motion Systems**: Ultra-smooth page transitions orchestrated with GSAP ScrollTrigger and Framer Motion spring physics.

## 🚀 Orbital Setup & Development

### 1. Prerequisite Installations
Ensure you have [Node.js](https://nodejs.org/) (v18+) installed on your machine.

### 2. Telemetry Ignition (Local Installation)
Clone the repository and install dependencies:

```bash
git clone https://github.com/Nithin2745/portfolio.git
cd portfolio
npm install
```

### 3. Launch Development Orbit
Run the local development server to test real-time interactions:

```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your web browser.

### 4. Quantum Compile (Production Build)
Compile and optimize assets for deployment:

```bash
npm run build
```
The optimized client build will be generated in the `dist/` directory.
